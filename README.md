# 🚀 Smart System

## ✨ Overview
نظام بسيط يستخدم الحساسات لقراءة البيانات من البيئة والتفاعل معها بشكل لحظي.

## 💡 Idea
تحويل قراءات الحساسات إلى استجابة واضحة ومباشرة.

## ⚙️ How it works
- الحساسات تجمع البيانات  
- المتحكم يعالجها  
- النظام يعطي مخرج (LED / تنبيه)

## 🛠️ Tools
- Arduino / ESP32  
- Sensors  
- Arduino IDE  

## 🎯 Features
- قراءة لحظية  
- تصميم بسيط  
- قابل للتطوير
